# 🌤️ Weather App

> Uma aplicação web dinâmica para consulta de previsão do tempo em tempo real, desenvolvida com foco no consumo de APIs RESTful.

🔗 **[Acesse a aplicação rodando aqui](https://gabrielbrq.github.io/weather-app/)**

## 💡 Sobre o Projeto
O Weather App foi construído com o objetivo principal de consolidar na prática o uso da **Fetch API** e o gerenciamento de requisições assíncronas no JavaScript (Async/Await e Promises). A aplicação permite que o usuário digite o nome de qualquer cidade e receba instantaneamente as condições climáticas atuais, demonstrando domínio na manipulação do DOM a partir de dados externos.

## ✨ Funcionalidades
- **Busca Dinâmica:** Pesquise o clima de qualquer cidade do mundo em tempo real.
- **Consumo de API:** Integração fluida com API externa de dados climáticos.
- **Tratamento de Respostas:** Gerenciamento de estados de carregamento e exibição de dados precisos vindos do formato JSON.

## 🛠️ Tecnologias Utilizadas
- **JavaScript (ES6+)**
- **Fetch API** (Async/Await)
- **HTML5 & CSS3**
- **Git & GitHub Pages** (Deploy)
